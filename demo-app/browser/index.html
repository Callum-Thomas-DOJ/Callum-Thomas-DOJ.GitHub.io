States

:hover state

By applying a .hover class to the buttons, we can simulate their style when the buttons are hovered over by the mouse. In addition to these colours, the cursor is a pointer type cursor.

:focus-visible state

By applying a .focus-visible class to the buttons, we can simulate their style when the buttons are focused via keyboard. Note the similarity to the hover state styles, but there is the additional focus-ring surrounding the button. As can be seen below, this is not always visible and relies on a lighter background colour.

:active state

By applying an .active class to the buttons, we can simulate their style when the buttons are in an :active state, usually when the mouse is actively clicking on the button. Note the similarity to the previous states with an inset box-shadow for a subtle visual effect.

disabled state

By applying the .disabled class to the <button> element, we can simulate the disabled state styles on the buttons. In this state, the user will be unable to click the button, and none of the other state styles will apply.

Properties

Our custom button variant uses the following CSS properties:

State Property CSS Property Value Comment
Text Colour color #194658 Used as the text colour of the buttons in TeamSite website.
Background Colour background-color #BCE2E7 Used as the background colour of the buttons in TeamSite website.

:hover Text Colour color #194658 Used as the hover state text colour of the buttons in TeamSite website.

:focus-visible Text Colour color #194658 Used as the focus state text colour of the buttons in TeamSite website.

:focus-visible Border Colour border-color #205D76 Used as the focus state border colour of the buttons in TeamSite website.
:focus-visible Outline outline 0 Set by Bootstrap 5.3 to remove any browser defaults.
:focus-visible Box Shadow box-shadow 0 0 0 0.25rem rgba(31, 90, 114, 0.5) Calculated and set by Bootstrap 5.3 using border and text colours.
:active Text Colour color #194658 Used as the active state text colour of the buttons in TeamSite website.
:active Background Colour background-color #C9DD6B Used as the active state background colour of the buttons in TeamSite website.
:active Border Colour border-color #205D76 Used as the active state border colour of the buttons in TeamSite website.
:active Box Shadow box-shadow inset 0 3px 5px rgba(0, 0, 0, 0.125) Inset box shadow set by Bootstrap 5.3. to give a 3D visual effect.
:disabled Text Color color #194658 Used as the disabled state text colour of the buttons in TeamSite website.

:disabled Box Shadow box-shadow none Removed by Bootstrap 5.3.
